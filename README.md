# AI-Based Food Waste Pattern Analyzer

This project is part of the AI for Sustainability Virtual Internship.

## Problem
College hostels waste large amounts of food due to poor planning and lack of data-driven decisions.

## Solution
An AI-based decision-support system that:
- Analyzes hostel food waste data
- Identifies waste patterns
- Generates recommendations to reduce food waste

## SDG Alignment
SDG 12 – Responsible Consumption and Production
